💡 Tip: Kopieer de HTML en plak het als HTML in je email client signature (niet als gewone tekst!)

Installeren in je email client

Gmail: Instellingen → Signature → Plak de HTML (gebruik een extensie zoals "Email Signature Decoder" of plak via "Show plain text" mode)

Outlook: File → Options → Mail → Signatures → Nieuwe signature → Plak in HTML editor

Apple Mail: Plaak de code als HTML in Mail → Preferences → Signatures

Snelste manier: Kopieer de visuele preview hierboven en plak direct in je email client als opgemaakte tekst
