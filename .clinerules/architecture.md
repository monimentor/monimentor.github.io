# Architecture and Tech Stack

This document outlines the architecture and technology stack for the Monimentor website.

## Architecture

The website is a single-page application (SPA) built with Vue.js. Vite is used as the build tool and development server. running on github pages under /docs

## Tech Stack

- **Framework:** Vue.js
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Package Manager:** npm
- **Development Environment:** devenv
